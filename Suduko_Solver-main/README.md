# 🧩 Sudoku Glow

A modern, responsive Sudoku game built using only **HTML, CSS, and JavaScript**.  
It includes animated floating number backgrounds, sound effects, a hint system, difficulty levels, timer, and move counter.

## ✨ Live Features

- 🔢 9x9 interactive Sudoku board
- 🟢 Easy, 🟡 Medium, and 🔴 Hard difficulty modes
- 💡 Hint system for stuck users
- ✅ Check, Solve, Reset, and New Game buttons
- ⏱️ Timer and move counter
- 🔊 Sound effects for input and completion
- 🌌 Animated floating numbers background
- 📱 Fully responsive design for mobile and desktop
- 🚫 No libraries and no backend required

## 👀 Preview

> A clean glass-style Sudoku game with glowing UI, floating number animation, and smooth gameplay.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. 📥 Download or clone the project files.
2. 🌐 Open `index.html` in any modern browser.
3. 🎮 Start solving the Sudoku puzzle.

## 🎯 How to Play

- 👆 Click any empty cell and enter a number from 1 to 9.
- 💡 Use **Hint** if you get stuck.
- ✅ Use **Check** to verify your current board.
- 👁️ Use **Solve** to reveal the full solution.
- 🔄 Use **Reset** to restart the current puzzle.
- 🆕 Use **New Game** to load a new puzzle.
